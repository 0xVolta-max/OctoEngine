# LiteLLM Adapter

LiteLLM provides model routing.

## Responsibilities

- expose role-based model aliases,
- abstract provider differences,
- support cloud and local models,
- enable per-agent model policies,
- record model usage where available.

## Recommended Aliases

- `opusplan`
- `implementer`
- `fast-cloud`
- `fast-local`
- `vision`
