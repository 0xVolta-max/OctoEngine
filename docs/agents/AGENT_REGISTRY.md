# Agent Registry

The Agent Registry lists official OctoEngine agents.

## Core Agents

| Agent                        | Purpose                                                |
| ---------------------------- | ------------------------------------------------------ |
| evolution-manager-agent      | Evolves Engineering OS and agent system design         |
| graphify-scope-agent         | Creates repository and dependency scope                |
| caveman-inspector-agent      | Performs targeted file inspection                      |
| obsidian-memory-agent        | Retrieves project memory and intent                    |
| frontend-taste-agent         | Reviews product feel and UX flow                       |
| impeccable-reviewer-agent    | Reviews visual polish and UI quality                   |
| emil-kovalski-director-agent | Reviews visual direction and cyber-brutalism alignment |
| qa-verification-agent        | Executes and reports verification ladder               |
| docs-compression-agent       | Updates concise documentation and handoff notes        |

## Agent Registration Fields

Every agent must define:

- id,
- display name,
- mission,
- responsibilities,
- inputs,
- outputs,
- tools,
- forbidden actions,
- escalation conditions,
- verification expectations.
