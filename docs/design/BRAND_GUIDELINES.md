# Brand Guidelines

## Name

OctoEngine

## Tagline Options

- The Open Source Agent Engineering Control Plane
- Build. Orchestrate. Verify. Ship.
- Agentic Engineering, Governed.
- The workflow engine for AI-native software delivery.

## Voice

Professional, precise, technical, confident.

Avoid hype.

Use clear language around safety, governance, verification, and extensibility.

## Brand Metaphor

OctoEngine is an engine with multiple arms:

- one arm scopes,
- one inspects,
- one plans,
- one implements,
- one reviews,
- one verifies,
- one documents,
- one ships.

## Visual Motifs

- graph nodes,
- industrial panels,
- terminal metadata,
- neon-lime system signals,
- dark grid backgrounds,
- agent activity trails,
- verification gates.
