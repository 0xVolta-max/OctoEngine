# North Star

A developer can run:

```text
/gsd-plan
```

or trigger the equivalent action in AionUI.

OctoEngine then:

1. retrieves relevant project memory,
2. builds a scope graph,
3. inspects only high-value files,
4. creates an implementation plan,
5. assigns specialized agents,
6. performs implementation through approved executors,
7. verifies the result,
8. updates documentation,
9. prepares a ship report.

## North-Star Outcome

The developer receives:

- a scoped plan,
- agent evidence,
- verification results,
- risks and known gaps,
- updated docs,
- next-action recommendation.

## Success Condition

OctoEngine is successful when it reduces the cost of safe, high-quality agentic software delivery.
