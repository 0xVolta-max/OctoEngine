# Roadmap

## v0.1 — Foundation

Goal: create the complete public repository foundation.

Deliverables:

- professional documentation set,
- agent contracts,
- workflow specs,
- workspace schemas,
- adapter specifications,
- realization prompts,
- example Basura integration.

## v0.2 — Executable Core

Goal: implement the first executable control-plane primitives.

Deliverables:

- TypeScript package scaffolding,
- agent registry loader,
- workflow parser,
- event bus,
- state model,
- CLI entrypoint,
- basic JSON workflow execution.

## v0.3 — AionUI Workspace Integration

Goal: expose OctoEngine state to a visual workspace.

Deliverables:

- graph-view JSON endpoint,
- kanban-state endpoint,
- activity event stream,
- workflow status model,
- AionUI adapter examples.

## v0.4 — MCP Integration Layer

Goal: support read-only MCP tool integrations.

Deliverables:

- Graphify adapter,
- Caveman adapter,
- Obsidian adapter,
- Memory adapter,
- GitHub adapter,
- policy guards.

## v0.5 — Runtime Integrations

Goal: connect orchestration to execution tools.

Deliverables:

- Hermes adapter,
- Claude Code launcher profiles,
- OpenCode launcher profiles,
- Codex integration docs,
- LiteLLM profiles.

## v1.0 — Open Agent Engineering Platform

Goal: stable open-source release for real project workflows.

Deliverables:

- stable agent registry,
- stable workflow engine,
- AionUI workspace support,
- CLI support,
- documentation site,
- examples,
- contribution framework.
