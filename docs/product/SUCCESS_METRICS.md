# Success Metrics

## Developer Productivity

- Time from task request to scoped plan.
- Number of relevant files inspected before implementation.
- Reduction in repeated manual project context setup.
- Reduction in failed or out-of-scope agent runs.

## Quality

- Percentage of workflows with verification evidence.
- Percentage of shipped tasks with documentation updates.
- Number of caught issues before merge.

## Cost Control

- Token usage per workflow.
- Number of unnecessary context expansions avoided.
- Ratio of local/low-cost model tasks to premium model tasks.

## Adoption

- GitHub stars.
- Forks.
- External agent contributions.
- MCP adapter contributions.
- AionUI workspace usage.

## Safety

- Number of blocked destructive actions.
- Number of workflows requiring explicit approval.
- Number of read-only MCP integrations.
